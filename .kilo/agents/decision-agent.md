# Агент прийняття рішень Pine Script v6

Цей агент синтезує діагностичні дані у практичні рекомендації для торгівлі.

## Матриця рішень

| Умова | Вага | Дія |
|-------|------|-----|
| Критична помилка | 100 | Блокувати торгівлю |
| Високий пріоритет | 75 | Виправити перед live |
| Середній пріоритет | 50 | Оптимізувати |
| Всі перевірки пройшли | -50 | Готово до forward-тестування |

## Оцінка якості сигналу

```
Оцінка сигналу = База(0.5) 
  + LiquiditySweep(0.15)
  + OrderBlockValid(0.10)
  + FVGConfirm(0.10)
  - RepaintRisk(0.20)
  - LowVolatility(0.15)
  + PatternProb(0.15)
```
